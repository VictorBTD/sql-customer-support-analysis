# Customer Support Data Analysis (SQL)

## Overview
This project analyzes customer support ticket data using SQL to identify trends, performance metrics, and operational insights.

## Objectives
- Analyze ticket volume by issue type
- Measure average resolution time
- Evaluate ticket distribution by priority
- Identify trends over time

## Dataset
The dataset simulates customer support tickets, including:
- Ticket ID
- Issue Type
- Priority
- Created Date
- Resolved Date
- Status

## Tools Used
- SQL (data querying and analysis)
- CSV dataset

## Key Analysis Performed
- Grouped tickets by issue type to identify most common problems
- Calculated average resolution time
- Analyzed ticket distribution by priority
- Evaluated monthly trends in ticket creation

## Key Insights
- Certain issue types occur more frequently than others
- Resolution times vary depending on ticket type and priority
- Ticket volume shows patterns over time
- Priority distribution helps identify workload intensity

## Files
- dataset/support_tickets.csv → dataset used
- queries/analysis_queries.sql → SQL queries used for analysis

## Notes
This is a practice project created to demonstrate SQL data analysis skills using a simulated dataset.
